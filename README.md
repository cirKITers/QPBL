# Quantum Principal Base Learning (qPBL)
## A framework to learn similar features under basis transformations

The general framework consists of a superposition module that encapsulates the superposition of different images with the same label, and a second training module that learns the different transformations. The optimization criterion is the interference for the same principal components extracted in the LAS module.
